# Notion + Nanoleaf

#### Get token

```
./node_modules/.bin/nanoleaves token
```

#### Set ENV vars:

```
HOST=192.168.1.217
BASE=/api/v1/
PORT=16021
ACCESS_TOKEN=?
```
